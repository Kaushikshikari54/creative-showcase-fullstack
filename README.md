# Creative Showcase - Fullstack Project

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Express.js (or Flask)
- **Database**: MongoDB (or your DB)

## Features
- User Authentication (Login/SignUp)
- Profile Management
- Image Upload & Gallery
- Responsive Design

## Setup & Run
### Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
npm start
